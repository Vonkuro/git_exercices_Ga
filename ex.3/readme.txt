Vous venez de réaliser que vous avez commité un fichier de backup : main.cpp~.

Les fichiers de backups ne doivent pas être commités, corrigeons cette erreur.

- Supprimer le fichier avec 'rm'
- Indéxer tous les changements, y compris les fichiers supprimés avec 'git add .'
- Ecrire un .gitignore qui corrige cette erreur.
- Indéxer ce fichier.
- Commiter
